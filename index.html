About Me

Hi. I'm Serena and I live in Seattle, WA.

I'm a full-stack developer at eventcore, where I use C#, ASP.NET MVC, jQuery, and SQL to make conference registration websites for thousands to tens of thousands of users.

I've always been interested in computers, but unlike most, I discovered my love for programming later in life. After an economics degree from the University of Washington and some underwriting/analyst jobs in the health insurance industry, I decided to change careers. In 2015, I went back to school to study computer science. I graduate from the University of British Columbia with a Bachelor of Computer Science in May 2018.

Projects

Blob Pets

An Android virtual pet game where you can adopt, raise, breed, exercise, and fight blobs near you. I did the front-end for this group project and created and coded the scenes for the user profile, blob rooms, exercising, and scoreboards. We designed and created a RESTful API. This project used Unity2D, C#, PHP, and Laravel.

AE Record Tracker

A record-management web app that was made to help Associated Engineering (AE) manage and retrieve paper files from their file rooms and/or offsite storage. For this group project, I was the project manager, did the back-end for quick search and record type specific search and the front and back-end for the CRUD interface to manage user permissions and locations. This project used Java, Spring framework, React, HTML/CSS, and Javascript.

Connie's Cookhouse

Mock up website for a restaurant.

NailStash

NailStash is an Android app that allows users to search for nail polish swatches (photos of nail polish painted on fingers) by scanning the product's barcode. It's common to Google swatch photos when buying nail polish in stores. This app automates that process. This project used Java, the Google Vision Barcode API, UPC Item DB API, and Google Custom Search JSON API.

Tweets 'Round The Corner

Tweets 'Round The Corner shows tweets within 1 km of your current location. This was the app built for nwHacks2018. The group project used Java, the Google Maps API, and the Twitter API.
